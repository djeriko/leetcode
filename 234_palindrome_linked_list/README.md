####***234. Palindrome Linked List***

Level - Easy

Given the head of a singly linked list, return true if it is a palindrome.

```
Input: head = [1,2,2,1]
Output: true
```

```
Input: head = [1,2]
Output: false
```

Constrains:
The number of nodes in the list is in range [1, 10].
0 <= Node.val <= 9
